2. Your First Session

  1. Open CrystalCare.
  2. Put on headphones. Headphones are recommended for all modes and essential for Taygetan Resonances mode.
  3. Select a frequency mode. If you are new, Lower Frequencies or Combined Mode are good starting points.
  4. Enter a session duration. 20 means 20 minutes. The field accepts flexible formats (see below).
  5. Select an output device if you have more than one. The dropdown refreshes automatically if you plug in or remove a device during the app's lifetime.
  6. Press Play.
  7. Close your eyes. Set an intention if you wish. Let her take over.

Sessions under 60 seconds produce the core tone only. Sessions of 60 seconds or longer activate all six sacred healing layers for the full experience. For deep work, 10–60 minutes is a common range, but she has no upper limit — you can run sessions of any length (hours, even overnight) at constant ~50 MB memory.

Duration Format

The Session Duration field accepts multiple formats:

  • 90 → 90 minutes (plain number is treated as minutes)
  • 1.5h → 1.5 hours = 90 minutes
  • 1h30m or 1h 30m → 1 hour 30 minutes
  • 1:30 → 1 hour 30 minutes (colon notation)
  • 1:30:45 → hours:minutes:seconds
  • 2h, 30s → bare units supported
  • 2h 15m 30s → combined units

Streaming playback has no upper limit. Saves are capped at 60 minutes due to RAM constraints.

3. The App Controls

Play

Streaming playback begins within a fraction of a second through the producer-consumer pipeline. Audio is generated in real time, 3-second chunks at a time, at constant memory usage. Double-precision time arithmetic maintains stable resonance through sessions of any duration, including 12+ hours.

Save as WAV

Generates the full session in memory and writes a 16-bit PCM WAV file. The full-session generation is computationally intensive — expect 40–55% CPU usage during generation, which is normal. Saves produce the same sound as streaming; they use the same 16-stage pipeline.

Because saves hold the full session in memory, they are RAM-bound:

System RAMRecommended Max Save Duration
16 GB15–20 minutes
32 GB30 minutes
64 GB+60 minutes

Streaming playback has no such limit. For longer saved files, save multiple shorter sessions.

Batch Save

Asks how many tones to generate and which folder to save them to, then produces that many WAV files (named tone1.wav, tone2.wav, etc.). Each file has a fresh random base frequency, so each one is unique.

Open Guide

Opens this guide in your default browser.

Stop

Cancels the current operation and stops playback.

Output Device

Select which audio device CrystalCare resonates through. "Default Output Device" uses your Windows default. The list updates automatically in real time when you plug in or remove a device — no need to restart the app.
